# Fondamenti di Programmazione 1 - UNICAL
Questa repository contiene le tracce e gli esercizi risolti presenti su domjudge per il corso di Fondamenti di Programmazione 1.

## Come contribuire
Negli anni gli esercizi sono cambiati, quindi se avete esercizi risolti che non sono presenti in questa repository, potete contribuire con una pull request.

Un obiettivo è quello di commentare il codice in modo da renderlo più comprensibile. Per questo motivo, se volete contribuire, vi chiedo di commentare il codice.
